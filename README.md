Solution Center
====================

ServiceNow Solution Center in Angular 1.5 and TypeScript.

## Running Local

1. Clone the repo

1. Run `npm i`

1. Run `typings install`

1. Run `bower install`

1. Run `npm start` to start the server and launch the app
